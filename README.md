# MeshTune — Калькулятор регулировки сетки кровати 3D-принтера

**MeshTune: Don't wing it. Screw it right.** — инструмент для точной калибровки выпрямления кровати на 3D-принтерах с механическими винтами (как Flashforge AD5X и им подобные).

## Что это?

MeshTune анализирует **карту выпрямления (bed mesh)** вашего принтера и рассчитывает ровно сколько оборотов (и в какую сторону) нужно закрутить каждый винт, чтобы выровнять кровать по плоскости.

Инструмент поддерживает файлы конфигурации:
- `printer.cfg` и `printer.base.cfg` (Klipper/Moonraker)
- `probe8899.txt` / логи сканирования (Flashforge AD5X)
- Любые текстовые файлы с mesh-данными

## Как использовать?

1. **Загрузить конфиг:** Нажмите кнопку "Choose File" и выберите один или несколько файлов (`.cfg`, `.txt`, `.log`)
   - Можно загружать одновременно `printer.cfg` и `printer.base.cfg` — инструмент выберет приоритетный источник автоматически

2. **Выбрать параметры:**
   - **Шаг винта (мм/оборот):** Выбрать из выпадающего списка (обычно 0.5–1.0 мм/оборот)
   - **Сопло:** Диаметр сопла экструдера (0.2, 0.4, 0.6, 0.8 мм)
   - **Допуск:** Целевой допуск выравнивания (по умолчанию 0.15 мм)

3. **Механика винта:**
   - **Режим:** "закрутить" (поднять кровать) или "открутить" (опустить)
   - **Направление:** "ближе к соплу" или "дальше от сопла"

4. **Просмотреть результаты:**
   - **Таблица расчёта винтов:** Для каждого угла кровати показаны дельта (мм), обороты, градусы, направление и действие
   - **Heatmap/Таблица:** Визуализация карты выпрямления
   - **Модель и метрики:** Остатки планарности (PV, RMS), скручивание, оценка

## Функции

### Экспорт

- **Копировать отчет:** Копирует текстовый отчет в буфер обмена (параметры, результаты расчёта)
  - Нажмите кнопку "Копировать отчет" → отчет скопирован (кнопка показывает зелёную галочку)
  - Вставьте его куда угодно (`Ctrl+V` / `Cmd+V`)

- **CSV:** Экспортирует таблицу винтов в CSV формат
  - Нажмите кнопку "CSV" → скачивается файл `MeshTune_YYYY-MM-DD.csv`
  - Откройте в Excel, Google Sheets или любом текстовом редакторе

### Визуализация

- **Heatmap (canvas):** Интерактивная тепловая карта проседаний кровати
- **Table (HTML):** Табличный вид всех Z-значений

### Сравнение (до/после)

Загрузите второй mesh-файл для сравнения результатов регулировки. Инструмент покажет улучшения в планарности и остатках.

## Установка

Просто откройте `frontend/mesh-tool.html` в браузере — всё работает локально, без подключения к серверу.

```bash
# Или запустить локальный веб-сервер:
python -m http.server 8000
# Затем откройте http://localhost:8000/frontend/mesh-tool.html
```

## Примеры файлов

### Flashforge AD5X
Экспортируйте результат сканирования: Settings → Maintenance → Bed Leveling → Export Report

### Klipper
Файл автоматически генерируется при запуске `BED_MESH_CALIBRATE` в `printer.cfg`

## FAQ

**Q: Какие файлы поддерживаются?**
A: `.cfg`, `.txt`, `.log` — любые текстовые файлы с mesh-данными

**Q: Можно ли загружать несколько файлов?**
A: Да, инструмент поддерживает загрузку `printer.cfg` + `printer.base.cfg` одновременно

**Q: Где хранятся мои данные?**
A: Всё обрабатывается локально в браузере. Никакие данные не отправляются на сервер.

**Q: Что означает "PV" и "RMS"?**
A: 
- **PV (Peak-to-Valley):** Максимальное отклонение от плоскости
- **RMS (Root Mean Square):** Среднеквадратичное отклонение
